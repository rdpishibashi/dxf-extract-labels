# DXF Extract Labels

DXFファイルからテキストラベルを抽出し、Excel形式で出力するStreamlitアプリケーションです。

## 機能

- **複数ファイル対応**: 複数のDXFファイルを一括処理
- **ラベル抽出**: DXFファイルからTEXT、MTEXTエンティティを抽出
- **機器符号（候補）抽出**: 図面枠内・図面情報欄外のラベルからReference Designator
  パターンに一致し除外パターンに該当しないものを対象にする。確実に機器符号と判定
  できる形（確定パターン）は自動採用され、それ以外は「未確定ラベル」として一覧
  表示し、採用するものを選択できる（チェックしたものだけが出力される）
- **図面番号抽出**: 図番と流用元図番を自動識別（ラベルベース・座標ベース）
- **タイトル・サブタイトル抽出**: 図面のタイトルとサブタイトルを自動抽出
- **複数図面対応**: 1ファイルに複数図面がある場合、全図面枠を対象
- **矩形領域抽出**: 図面枠内の矩形（直交ポリゴン）領域を検出し、領域内ラベルに領域名を付与（「領域選択オプション」）
- **Excel出力**: 統計情報とラベル一覧をExcel形式で出力
- **自動集計**: ファイルごとの個別シートとサマリーシートを生成

## 使用方法

### 基本的なワークフロー

1. **ファイルのアップロード**: 複数のDXFファイルをアップロード可能
   - ドラッグ&ドロップまたはファイル選択
   - 複数ファイルの同時アップロード対応

2. **オプション設定** (任意):
   - **機器符号（候補）以外も抽出**: OFF（既定）では図面枠内・図面情報欄外の
     ラベルのうち機器符号パターンに一致し除外パターンに該当しないものが対象。
     確実に機器符号と判定できる形（確定パターン）は自動採用され、それ以外は
     「未確定ラベル」として一覧表示し、採用するものを選択する。
     ON では図面枠制限・フィルタなしで全ラベルを抽出する（従来動作）
     - 候補パターン例: `CNCNT`, `FB`（英字のみ）、`R10`, `CN3`, `AAC1B4-07`（英字+数字+ハイフン任意）、
       `CN-IF2-1`（英字-英字+数字）、`FB()`, `MSS(MOTOR)`, `R10(2.2K)`（括弧付き、括弧より前で判定）
   - **図面番号抽出**: 図番と流用元図番を自動識別
   - **並び替え**: 昇順、降順、並び替えなしを選択

3. **ファイル処理**: 「ラベルを抽出」ボタンをクリックして抽出開始

4. **未確定ラベルの選択** (機器符号（候補）以外も抽出＝OFF時のみ):
   - ファイルごとに、機器符号パターンに一致し除外パターンに該当しなかったラベルの
     うち、確定パターンにも一致しなかったもの（一覧・個数、横2列表示）が表示される
     （確定パターンに一致したラベルはここに表示されず、自動的に採用される）
   - Reference Designator として採用するものにチェックを入れる（初期状態は全てOFF）
   - 「選択完了」をクリックすると、チェックしたラベルと確定分がExcelに出力される

5. **結果のダウンロード**: Excelファイルをダウンロード
   - ダウンロード後は「新しい抽出を開始」ボタンが青色になります
   - 「新しい抽出を開始」をクリックすると、アップロード済みファイルがクリアされます
     （オプション設定は保持されます）

### 出力ファイル

デフォルトのExcelファイル名: `extracted_labels.xlsx`

Excelファイルの構成（既定モード）:
- **Summaryシート**: 全ファイルの抽出結果概要
  - ファイル名、図面枠内ラベル数、機器符号（候補）数、未確定ラベル数（未採用）
  - 図番、流用元図番（図面番号抽出が有効な場合）
  - タイトル、サブタイトル（抽出された場合）
- **各ファイルシート**: 個別ファイルの機器符号一覧（確定パターン自動採用分＋未確定ラベルUIで採用したもの）

「機器符号（候補）以外も抽出」ON 時は、フィルタなしの全ラベルが同様の構成で出力されます。

### 矩形領域抽出（領域選択オプション）

図面枠（太さ100の線で囲まれた枠）内の矩形領域を検出し、領域内ラベルに領域名を付与します。

1. （任意）「領域検出の詳細設定」で検出パラメータを調整し「設定を保存」
2. 「領域を検出」をクリック
3. 「領域の確認」で各領域の名称候補をチェックボックスで選択（各領域の📐角座標も確認可能）
   - 「図面番号・タイトル・サブタイトルを抽出」が有効な場合、ファイル名の直下に
     「図番：… / タイトル：… / サブタイトル：…」が表示されます
4. 「ラベルを抽出」→「Excelをダウンロード」（`領域` 列・`領域一覧` シート付き）
   - `領域一覧` シートの `ページ No.` はファイル内の図面枠の通し番号です（DXF図番ではありません）
   - `領域別ラベル一覧` シートでは、検出した領域名ごとに全ファイル横断でラベルと個数を確認できます
     （同名の領域は複数ファイルにまたがって合算されます）
